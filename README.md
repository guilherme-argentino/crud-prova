# Crud prova
Prova de apitidão para consultoria

# Construindo
Na pasta que baixar estes fontes, execute:

```
$ mvn install
```

# Executando
Após construir o projeto, execute:
```
$ mvn spring-boot:run
```
ou
```
$ java -jar target/crud-prova-1.0.0.jar
```
